# 🚀 Інструкція з розгортання обчислювальної платформи (Laravel + Python/MPI)

Цей репозиторій містить гібридне середовище, розроблене для запуску наукових обчислень та вебдодатків. Під капотом працює зв'язка з вебфреймворку **Laravel 8** та середовища **Python 3** (з підтримкою MPI, NumPy, SciPy та Numba для важких розрахунків). Уся інфраструктура запакована в Docker, що гарантує її однакову роботу на будь-якому комп'ютері (включно з Mac на процесорах Apple Silicon).

## 🛠 Попередні вимоги
Перед початком роботи переконайтеся, що на вашому комп'ютері встановлено:
1. **Git** (для завантаження коду).
2. **Docker** та **Docker Compose** (рекомендується встановити Docker Desktop).

---

## 📦 Крок 1. Завантаження проєкту та запуск контейнерів

1. Відкрийте термінал і зклонуйте цей репозиторій на свій комп'ютер:
   ```bash
   git clone <ПОСИЛАННЯ_НА_ВАШ_РЕПОЗИТОРІЙ>
   cd <НАЗВА_ПАПКИ_ПРОЄКТУ>
   ```

2. Запустіть збірку та ініціалізацію Docker-контейнерів:
   ```bash
   docker-compose up -d --build
   ```
   > **⚠️ УВАГА:** Перша збірка може зайняти **10–15 хвилин**! У цей час Docker компілює системні бібліотеки (MPI, C++ компілятори) та наукові пакети Python. Якщо термінал здається "завислим" — це нормально, просто дочекайтеся завершення. Наступні запуски відбуватимуться за кілька секунд.

3. Перевірте, чи всі контейнери успішно працюють:
   ```bash
   docker-compose ps
   ```
   Ви маєте побачити 4 контейнери зі статусом `Up` (nginx_calculation, php_calculation, mysql_calculation, phpmyadmin_calculation).

---

## ⚙️ Крок 2. Встановлення Laravel

Оскільки ми використовуємо чистий контейнер, нам потрібно розгорнути сам фреймворк Laravel.

1. Увійдіть всередину працюючого PHP-контейнера:
   ```bash
   docker exec -it php_calculation bash
   ```
   *(Після цієї команди вигляд вашого термінала зміниться — ви тепер "всередині" віртуальної машини у папці `/var/www/html`)*.

2. Встановіть Laravel 8 через Composer. Оскільки директорія може бути не повністю порожньою (через приховані файли git), використовуємо безпечний метод через тимчасову папку:
   ```bash
   composer create-project --prefer-dist laravel/laravel:^8.0 tmp_app
   cp -r tmp_app/* tmp_app/.* . 2>/dev/null || true
   rm -rf tmp_app
   ```

3. Надайте правильні права доступу вебсерверу, щоб Laravel міг зберігати кеш та файли:
   ```bash
   chown -R www-data:www-data storage bootstrap/cache
   ```

## 🗄 Крок 3. Налаштування бази даних

База даних MySQL вже працює в окремому контейнері. Нам потрібно лише "познайомити" її з Laravel.

1. Відкрийте файл `.env` (він з'явився у вашій папці `code` на комп'ютері) у будь-якому редакторі коду.
2. Знайдіть блок налаштувань бази даних і приведіть його до такого вигляду:
   ```env
   DB_CONNECTION=mysql
   DB_HOST=mysql
   DB_PORT=3306
   DB_DATABASE=calculation_gen
   DB_USERNAME=calculation_gen
   DB_PASSWORD=form_gen
   ```

---

## 🔐 Крок 4. Налаштування системи авторизації

Ми використовуємо пакет Laravel UI з інтеграцією стилів Bootstrap для створення сторінок входу та реєстрації без ускладнення інфраструктури (без Node.js).

1. Перебуваючи в терміналі контейнера (`docker exec -it php_calculation bash`), виконайте:
   ```bash
   composer require laravel/ui
   php artisan ui bootstrap --auth
   ```

2. Згенеруйте таблиці в базі даних:
   ```bash
   php artisan migrate
   ```

3. **Підключення стилів:** Відкрийте файл `resources/views/layouts/app.blade.php` у вашому редакторі коду. 
   Знайдіть рядки, де підключаються `app.css` та `app.js` (у секції `<head>`), і **замініть їх** на прямі посилання Bootstrap CDN:
   ```html
   <link href="[https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css](https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css)" rel="stylesheet">
   <script src="[https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js](https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js)" defer></script>
   ```

## 🎉 Крок 5. Перевірка роботи

Тепер усе готово! Відкрийте браузер на своєму комп'ютері:

* **Головна сторінка порталу:** [http://localhost](http://localhost) (у правому верхньому куті будуть кнопки Login та Register).
* **Управління базою даних (phpMyAdmin):** [http://localhost:8080](http://localhost:8080)
  * *Сервер:* `mysql`
  * *Користувач:* `root`
  * *Пароль:* `calculation_gen`

## 💡 Запуск Python-скриптів

Ваш науковий код (наприклад, `main.py` з розрахунками матриць та гамільтоніанів) має лежати в папці `code`. Для його виклику з Laravel (наприклад, у контролері після натискання кнопки користувачем) використовуйте таку конструкцію:
```php
$output = shell_exec('python3 /var/www/html/main.py 2>&1');
```

---

## 🛑 Зупинка роботи
Щоб коректно зупинити сервери без втрати даних, виконайте у терміналі (на своєму комп'ютері, не в контейнері):
```bash
docker-compose stop
```
Якщо ви хочете повністю видалити контейнери (дані бази даних збережуться у папці `db_data`), виконайте:
```bash
docker-compose down
```
